## Class-Attendance
- Create your database (XAMPP Recommended)
- Name your database `attendance`
- Create a table and name it `register`
Use the following IDs in your columns so it can rhyme
with the data in the project
- 1st Column `regno`
- 2nd Column `email`
- 3rd column `course`
- 4th column `idno`
- 5th column `passwd`
